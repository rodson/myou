设置更新网络条件

public static void setUpdateOnlyWifi(boolean updateOnlyWifi)

updateOnlyWifi 布尔值true(默认)只在wifi环境下检测更新，false在所有网络环境中均检测更新。特别提示：针对机顶盒等可能不支持或者没有无线网络的设备，请同样将该参数设为false。

只有当更新回调显示检测到更新的时候，可以调用该函数下载新版本。

public static void startDownload(Context context, final Update update, File file, final MengYouOSDownloadListener listener)
