导入SDK

使用Android Studio导入SDK

1. maven仓库导入

在项目的build.gradle中配置maven仓库

allprojects {
    repositories {
        jcenter()
        //这里是 萌友sdk 的包仓库
        maven {
            url "http://mvn.gz.cvte.cn/nexus/content/repositories/releases"
        }
    }
}

在app目录下的build.gradle中配置依赖

dependencies {
    compile 'com.cvte.myou:os-services:0.3.5@aar'
}

2. 本地导入

下载Android系统 SDK，将下载的aar文件放到工程的app/libs目录，并将aar文件命名为myou-version.aar(如myou-0.9.0.aar)

在app目录的build.gradle文件中添加repositories配置，并指定依赖（依赖的包名可以任意，名字和版本号需要与aar文件的名字对应）

repositories {
    flatDir {
        dirs 'libs'
    }
}

dependencies {
    compile 'com.cvte:myou:0.9.0@aar'
}
